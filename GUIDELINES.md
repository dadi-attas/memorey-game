# הנחיות בנייה — חבילת משחקי המשחקייה

קובץ זה מרכז את כל הקונבנציות, הכללים והבקשות המצטברות של המשתמש.
להשתמש כזיכרון/מדריך לכל פיתוח משחק חדש או תיקון קיים.

---

## מטרה
חבילת משחקים מקומיים (HTML יחיד לכל אחד) שפועלים בלי אינטרנט על טאבלט Android לנדסקייפ.
קהל יעד: מבוגרים, כולל בני 80+. העיצוב נדיב, הכפתורים גדולים, הצלילים עדינים.

## מבנה החבילה

```
memorey-game/
├── index.html              ← התפריט הראשי ("המשחקייה")
├── games.js                ← רשימת כל המשחקים (רגיסטר)
├── manifest.json           ← הגדרות PWA
├── service-worker.js       ← מטמון לעבודה אופליין
├── icon-192.png            ← אייקון PWA (192x192)
├── icon-512.png            ← אייקון PWA (512x512)
├── GUIDELINES.md           ← המסמך הזה
└── <game>.html (×21)       ← קובצי המשחקים העצמאיים
```

**כל משחק הוא HTML יחיד עצמאי** — כל ה-CSS וה-JS inline. ללא תלויות חיצוניות.

### התפריט (`index.html`)
- כותרת "המשחקייה" + כפתור "יציאה" אדום בראש.
- 6 קטגוריות צבעוניות עם תווית, וגריד של כרטיסי משחקים בצבע הקטגוריה.
- לחיצה על כרטיס → ניווט לקובץ המשחק.
- לחיצה על "יציאה" → דיאלוג "לצאת מהמשחקייה?" (כן/לא). "כן" → פרידה → `window.close()` (וב-PWA fallback ל-about:blank).

### מעבר בין משחק לתפריט
- בתוך משחק, כפתור "סיום" → דיאלוג "לסיים את המשחק?" (כן/לא).
- "כן" → מסך פרידה 👋 "להתראות!" + `location.href = 'index.html'` אחרי 1.5 שניות.
- אין יותר `window.close()` בתוך משחקים בודדים.

## מנגנוני משחק אחידים

### רמות
- **3 הצלחות רצופות** → עלייה לרמה הבאה + מודאל "רמה חדשה!" + פאנפר.
- **5 טעויות רצופות** → ירידה אוטומטית ו**שקטה** רמה אחת (אין מודאל/הודעה).
- **רמה מקסימלית** → ממשיכים לנצח, הקאונטר הופך ל"ניקוד: X".

### ב-bubbles ו-stop (משחקי תגובה)
- **רמות נסתרות** — לא מוצגות למשתמש.
- ב-bubbles — **5 פספוסים רצופים** = דיאלוג "להמשיך?" (לא ירידת רמה).

### חוסר פעילות
- **12 שניות** בלי לחיצה/תנועה → דיאלוג "להמשיך לשחק?" (כן/לא).
- **10 שניות** ללא בחירה → סיום אוטומטי → חזרה לתפריט.

### כפתור "סיום"
- בצד אחד של הכותרת, אדום.
- לחיצה → דיאלוג "לסיים את המשחק?" (כן/לא).
- "כן" → מסך פרידה 👋 "להתראות!" + `location.href = 'index.html'` אחרי 1.5 שניות.

### כפתור "איפוס"
- חזרה מלאה לרמה 1, איפוס כל הקאונטרים.

## צלילים (Web Audio API)

### אתחול עם priming (תוקן בכל 21 המשחקים)
```js
let audioCtx = null;
const AC = window.AudioContext || window.webkitAudioContext;
if (AC) {
  audioCtx = new AC();
  // Silent oscillator to "wake up" the context before first real tone:
  try {
    const _o = audioCtx.createOscillator();
    const _g = audioCtx.createGain();
    _g.gain.value = 0;
    _o.connect(_g).connect(audioCtx.destination);
    _o.start();
    _o.stop(audioCtx.currentTime + 0.002);
  } catch(_) {}
}
```
הפריימר פותר את הבאג שהצליל הראשון במשחק לא נשמע.

### סוגי צלילים
- **הצלחה**: `sine C5 (523) → E5 (659)`, 0.13–0.14 שנייה, volume 0.16
- **שגיאה**: `sawtooth 260Hz → 150Hz`, 0.3 שנייה, volume 0.13
- **מעבר רמה**: 4 צלילים סינוס עולים (C→E→G→C), 0.18–0.2 שנייה

## PWA — הפעלה כאפליקציה בטאבלט

### קבצים נדרשים
- **`manifest.json`** — `display: fullscreen`, `start_url: index.html`, מצביע על אייקונים.
- **`service-worker.js`** — Cache-first, מטמון של כל הקבצים ב-`FILES_TO_CACHE`. **יש להעלות את `CACHE_VERSION`** בכל עדכון משחק.
- **`icon-192.png`, `icon-512.png`** — אייקוני PWA.

### התקנה
1. להעתיק את כל התיקייה לטאבלט (דרך USB / Google Drive / דוא"ל).
2. לפתוח את `index.html` בדפדפן. אם מוגש דרך HTTP(S), ה-service-worker יירשם ויאחסן לעבודה אופליין. מ-`file://` אין service-worker אבל הכל עדיין עובד כי הכל מקומי.
3. "הוסף למסך הבית" → מקבלים אייקון נפרד ומסך מלא.

## העדפות משתמש

- **שפה**: עברית בלבד. אימוג'ים להצגת דמויות/אובייקטים.
- **פתגמים** (ב-proverbs.html): **רק פתגמים עבריים/ישראלים מקוריים**, לא תרגומים. מאגר של 41 פתגמים מאושר.
- **בפאזל**: חלקי Jigsaw עם "זכר ונקבה" תואמים מושלמים.
- **במבוך (maze.html)**:
  - עכבר 🐭 + גבינה 🧀.
  - רוחב מסלול ויזואלי 55–120 פיקסלים + אזור סובלנות 75–140.
  - **תמיד** אזור סובלנות > מסגרת ויזואלית.
  - מרחק עקיבה של העכבר: 90 פיקסלים מאחורי האצבע (lerp חלק).
  - הגבינה נאכלת רק כשהמשתמש מרים את האצבע (או fallback של 5 שניות).
  - התחלה: רצף 1.1 שניות עם הבהוב עכבר + "פיפ פיפ" + טקסט "צא לדרך!".
